# Person-Portfolio-Project
This is my personal portfolio website created to showcase my background, skills, and projects. It is built using HTML, CSS, and JavaScript, and highlights my learning journey and interest in web development.

## Tech Stack Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages (for deployment)

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any modern web browser
3. No additional setup or dependencies required

## Key Decisions & Challenges
- Focused on a clean and simple layout to keep the content easy to read
- Implemented basic animations and transitions using CSS
- Adjusted styles to make the website responsive for mobile devices
- Learned how to deploy and manage updates using GitHub Pages
